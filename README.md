# biochatter-server

This repository contains the Flask app to provide a RESTful API for [BioChatter](https://github.com/biocypher/biochatter). The app is also available as a Docker image from [Dockerhub](https://hub.docker.com/orgs/biocypher/repositories) (under the biocypher/biochatter-server repository). It is used in the [ChatGSE Next](https://github.com/biocypher/chatgse-next) web application.
